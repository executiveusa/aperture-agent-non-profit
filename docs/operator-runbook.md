# APERTURE Operator Runbook

1. Open Mission Control (`/overview`).
2. Create or verify an environment in `/environments`.
3. Create a job in `/jobs` for one of the guard skills.
4. Add recurring schedule in `/schedules` via API.
5. Review runs in `/runs` and inspect evidence in `/evidence`.
6. If failure appears real, create or update issue in `/issues`.
7. Approve or reject risky actions from `/approvals`.
8. Promote screenshot baselines from `/baselines` after validating intended changes.
9. Run eval suite from `/evals` before release gate decision.
