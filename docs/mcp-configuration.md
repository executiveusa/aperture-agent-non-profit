# MCP Configuration Notes

## Context7 MCP
Set:
- `CONTEXT7_MCP_URL`
- `CONTEXT7_MCP_API_KEY`

## OpenAI Docs MCP
Set:
- `OPENAI_DOCS_MCP_URL`
- `OPENAI_DOCS_MCP_BEARER`

## JCodeMunch (optional)
Set:
- `JCODEMUNCH_URL`
- `JCODEMUNCH_API_KEY`

If JCodeMunch is not configured, coding flows must fall back to local repository analysis only.
